IMC-M

 

Icas,   have a  series of micro Fire Panels for use with  different   detectors  and  in different applications.

 

  • IMC-Me, is a Fire Panel with 2 conventional zones, used together with our Chor-E optical smoke detector. This is a solution done for smaller applications i.e. appartments,private houses,common area's in smaller appartment buildings, Kindergardens, Pre-schools etc. The panel have a function which can activate the Sounders in all the installed detectors in one loop if a Fire Alarm is activated.
  • IMC-M.12, is a fire panel for 12v supplied detectors, i.e.Icas 300 series detectors.
